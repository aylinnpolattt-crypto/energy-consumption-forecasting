# Short-Term Energy Consumption Forecasting

[Open in Google Colab](https://colab.research.google.com/drive/1BhiiUYMvD78rXI9fTH0il2VNZZDQWEaw?usp=sharing)

This project was developed as part of a data science assignment to predict short-term energy consumption using historical data.

## Project Overview
- Task: Time-series forecasting  
- Models: Naive baseline and Random Forest  
- Evaluation: MAE and RMSE  

## What I did
- Prepared and cleaned time-series data  
- Created lag-based features to capture past behavior  
- Built a baseline model for comparison  
- Trained a Random Forest model  
- Evaluated performance using MAE and RMSE  

## Results
- Baseline MAE: ~62  
- Random Forest MAE: ~22  
- The model significantly improved prediction accuracy  

## Key Observations
- Energy consumption shows clear patterns over time  
- Random Forest captured these patterns better than the baseline  
- Errors increased during sudden changes in consumption  

## Tools
Python, Pandas, scikit-learn, Time Series Analysis
